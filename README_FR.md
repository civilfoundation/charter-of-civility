# La Charte de la Civilité

La Fondation Civile

## Aperçu

La Charte de la Civilité établit les principes directeurs, les engagements éthiques et les fondations structurelles de La Fondation Civile.
Elle sert de cadre de référence pour le discours public, le développement institutionnel et la coopération civique à l’ère contemporaine.

Ce document est publié de manière ouverte afin d’assurer la transparence, l’accessibilité et un dialogue continu entre les communautés, les disciplines et les juridictions.
